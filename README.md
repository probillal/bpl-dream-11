🏏 BPL Dream 11

BPL Dream 11 is a fantasy cricket web application where users can build their dream team by selecting players within a limited budget. The app provides an interactive and engaging experience similar to real fantasy league platforms.

🚀 Live Demo

🔗 https://superlative-crostata-d55052.netlify.app/

📂 GitHub Repository

🔗 https://github.com/probillal/bpl-dream-11

✨ Features

* Select players to build a fantasy team
* Budget (coin) based player selection system
* Manage and view selected team
* Interactive and responsive UI
* Mobile-friendly design
* Real-time feedback using notifications
  
🛠️ Tech Stack

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6+)
* React Toastify

⚙️ Installation & Setup

# Clone the repository

git clone https://github.com/probillal/bpl-dream-11.git

# Navigate to project directory

cd bpl-dream-11

# Install dependencies

npm install

# Run the project

npm run dev
